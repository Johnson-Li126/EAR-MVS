# EAR-MVS

This repository contains the official implementation of **EAR-MVS**:  
**EAR-MVS: Edge-Adaptive Refinement via Bimodal Gaussian Residual for Multi-view Stereo Reconstruction in Remote Sensing**  
**Authors**: Xiangrong Zhang, Junxiao Li, Feixiang Liu, Lianchao Zhang and Licheng Jiao

> The complete codebase, including pretrained weights, will be released upon acceptance of the paper.
